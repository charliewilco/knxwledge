# Reverse Polish Notation

Goal
- Evaluate postfix expressions using a stack.

Files
- Implementation: `src/ds/eval-reverse-polish-notation.ts`
- Tests: `src/ds/eval-rp-notation.test.ts`

Complexity
- O(n) time, O(n) space

Edge cases
- Invalid expressions throw

Example
- "2 3 + 4 *" -> 20.

Notes
- See the tests for expected behavior and additional scenarios.
