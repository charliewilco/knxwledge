# Stack and Queue

Goal
- Implement stack (LIFO) and queue (FIFO) using nodes.

Files
- Implementation: `src/ds/stack-queue.ts`
- Tests: `src/ds/stack-queue.test.ts`

Complexity
- Queue add/remove/peek: O(1)

Edge cases
- Peek/remove on empty queue throws

Example
- Queue add(1), add(2) -> remove() returns 1, then 2.

Notes
- See the tests for expected behavior and additional scenarios.
