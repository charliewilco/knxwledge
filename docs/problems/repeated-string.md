# Repeated String

Goal
- Count occurrences of 'a' in a repeated string prefix.

Files
- Implementation: `src/problems/repeated-string.ts`
- Tests: `src/problems/repeated-string.test.ts`

Complexity
- O(n) time relative to input string length

Edge cases
- Limit shorter than string length

Example
- "aba", n=10 -> 7.

Notes
- See the tests for expected behavior and additional scenarios.
