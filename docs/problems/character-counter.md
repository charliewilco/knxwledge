# Character Counter

Goal
- Count letter frequency in a sentence.

Files
- Implementation: `src/problems/character-counter.ts`
- Tests: `src/problems/character-counter.test.ts`

Complexity
- O(n) time, O(1) extra space (bounded alphabet)

Edge cases
- Ignores numbers and casing

Example
- "Hello" -> { h:1, e:1, l:2, o:1 }.

Notes
- See the tests for expected behavior and additional scenarios.
