# Move Zeros

Goal
- Move all zeros to the end while preserving order.

Files
- Implementation: `src/problems/move-zeros.ts`
- Tests: `src/problems/move-zero.test.ts`

Complexity
- O(n) time, O(1) extra space for in-place variant

Edge cases
- All zeros or no zeros

Example
- [1,0,2,0,3] -> [1,2,3,0,0].

Notes
- See the tests for expected behavior and additional scenarios.
