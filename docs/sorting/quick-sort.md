# Quick Sort

Goal
- Sort via partitioning around a pivot.

Files
- Implementation: `src/sorting/quick-sort.ts`
- Tests: `src/sorting/quick-sort.test.ts`

Complexity
- Average O(n log n), worst O(n^2)
- O(log n) stack space (in-place)

Edge cases
- Already sorted or reverse-sorted input can be worst-case

Example
- [9, 3, 7, 1] -> [1, 3, 7, 9].

Notes
- See the tests for expected behavior and additional scenarios.
