# Radix Sort

Goal
- Sort by grouping elements by digit position.

Files
- Implementation: `src/sorting/radix-sort.ts`
- Tests: `src/sorting/radix-sort.test.ts`

Complexity
- O(nk) time, O(n + k) space

Edge cases
- Requires a numeric key function

Example
- [170, 45, 75] -> [45, 75, 170].

Notes
- See the tests for expected behavior and additional scenarios.
