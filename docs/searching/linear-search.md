# Linear Search

Goal
- Scan an array for a target value.

Files
- Implementation: `src/searching/linear.ts`
- Tests: `src/searching/linear.test.ts`

Complexity
- O(n) time, O(1) space

Edge cases
- Empty array returns -1

Example
- ["a","b","c"], target "b" -> index 1.

Notes
- See the tests for expected behavior and additional scenarios.
