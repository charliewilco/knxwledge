# Array Flat

Goal
- Flatten nested arrays iteratively and recursively.

Files
- Implementation: `src/javascript/flat.ts`
- Tests: `src/javascript/flat.test.ts`

Complexity
- O(n) time, O(n) space

Edge cases
- Depth parameter limits recursion

Example
- flatRecursive([1,[2,[3]]], 1) -> [1,2,[3]].

Notes
- See the tests for expected behavior and additional scenarios.
